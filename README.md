# YouTube

Репозиторий содержит материалы, связанные с сетевой обработкой трафика сервиса YouTube.  
Здесь собраны данные и файлы, которые могут использоваться при построении правил маршрутизации, фильтрации, сегментации и сетевой политики.

## Содержимое репозитория

- [YouTube_RouterOS.rsc](YouTube_RouterOS.rsc) - файл с командами для роутеров MikroTik, позволяющий обновлять `address list` сервиса
- [YouTube_CIDR.txt](YouTube_CIDR.txt) - список IP-адресов и CIDR-диапазонов сервиса
- [YouTube_DNS](YouTube_DNS) - список доменных имён сервиса

## Назначение

Материалы репозитория могут использоваться для:

- подготовки и сопровождения `address list`
- построения правил маршрутизации и фильтрации трафика
- применения сетевых политик в рабочей или домашней инфраструктуре
- анализа сетевого взаимодействия отдельных сервисов
- автоматизации обработки данных на устройствах MikroTik

## Примечание

Материалы данного репозитория опубликованы исключительно в образовательных, исследовательских и административных целях.  
Они предназначены для изучения принципов сетевой обработки трафика, автоматизации RouterOS и управления доступом к сервисам в рабочей или домашней сети.
